🎯 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

METHOD 1: FULLY AUTOMATED (Recommended)
─────────────────────────────────────────────────────────────────────────────

In GitHub Copilot Chat, type:

    Run TestSpirit and fix all errors

That's it! Everything happens automatically:
  1. ✅ TestSpirit analysis runs
  2. ✅ Issues are prioritized
  3. ✅ Fixes are applied (using REAL project code)
  4. ✅ Verification runs
  5. ✅ Summary is provided


METHOD 2: CHAT PARTICIPANT
─────────────────────────────────────────────────────────────────────────────

In GitHub Copilot Chat:

    @testspirit /run_testspirit_fix


METHOD 3: MANUAL TASK
─────────────────────────────────────────────────────────────────────────────

1. Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac)
2. Type: Tasks: Run Task
3. Select: Run TestSpirit
4. Wait for .testspirit_output.json
5. Tell Copilot: "Fix all issues in .testspirit_output.json"


METHOD 4: SHELL SCRIPT
─────────────────────────────────────────────────────────────────────────────

From project root:

    sh testspirit-run.sh

Or with instructions:

    sh testspirit-fix-auto.sh

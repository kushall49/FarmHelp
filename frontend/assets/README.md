# Assets Folder

## Background Image Instructions

To add a background image to your home screen:

1. Find or create a farming-themed background image (recommended size: 1920x1080 or higher)
2. Save it as `background.jpg` in this folder
3. Alternatively, you can use PNG format: `background.png`

## Recommended Image Sources:
- Unsplash.com (search for "farm field", "agriculture", "green field")
- Pexels.com (free stock photos)
- Pixabay.com (free images)

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 1920x1080 or larger
- File size: Under 2MB for better performance
- Theme: Farming, agriculture, green fields, crops

- background.png ✅ (alternative)
